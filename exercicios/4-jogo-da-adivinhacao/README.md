# Exercício 4 - Jogo da Advinhação

Neste jogo, o objetivo é adivinhar um número aleatório gerado pelo computador. O número aleatório está dentro do intervalo de 1 a 100. O jogador tenta adivinhar o número com o menor número possível de tentativas.

## Funcionamento

1. O computador gera um número aleatório entre 1 e 100.

2. O jogador faz um palpite sobre qual é o número.

3. O programa fornece feedback ao jogador, informando se o palpite está "acima", "abaixo" ou se ele adivinhou o número.

4. O jogador continua fazendo palpites até adivinhar corretamente o número gerado pelo computador.

5. Após o jogador adivinhar o número, o programa exibe o número de tentativas que o jogador levou para acertar.

## Como Jogar

1. Execute o programa.

2. O computador irá gerar um número aleatório entre 1 e 100.

3. Insira um palpite sobre qual é o número.

4. O programa fornecerá feedback sobre o seu palpite.

5. Com base no feedback, faça outro palpite.

6. Continue fazendo palpites até adivinhar corretamente o número.

7. Após adivinhar o número, o programa informará quantas tentativas você levou para acertar.
